Exercise 1 - Examine Patching, Upgrading and Lifecycle Management

In this exercise, you will review the methodologies of applying patches, upgrades, and deployments we use to manage an environment. You will also manage backups and the Lifecycle of data, including solutions for long-term retention as well as methods to determine the frequency and quality of these backups. 

You will also learn the details of backups and the Lifecycle process of methodologies used to determine how frequently to perform backups and how much data to retain. You will review different resources typically patched in a cloud environment that include hypervisors, VMs, virtual appliances, applications, storage, operating systems, and rolling back in the event of any issues that arise.

You will also cover upgrade methods and deployment methodologies used in the field, such as rolling upgrades, blue-green, and canary.

Task 2 - Update and Upgrade Ubuntu

Now that the Ubuntu VM has been created and installed, we will explore the upgrading and patching process.

In this task, you will upgrade Ubuntu Server 22.04 LTS.

Step 1

Before performing upgrades to the new VM, firstly the proxy will need to be updated in order to allow the connection through the lab environment.

Type the following command:

sudo nano /etc/apt/apt.conf.d/proxy.conf

Press Enter.

Step 2

This will create and open the proxy configuration file.

Step 3

In the new file, type the following:

Acquire::http::Proxy "http://192.168.255.13:8080/";

Acquire::https::Proxy "http://192.168.255.13:8080/";

Step 4

To save and close the configuration file, execute the Control + O keyboard shortcut, followed by the Control + X keyboard shortcut.

The prompt will return at the foot of the window.

Step 5

From the command line type:

sudo apt-get update

Press Enter.

If you are asked to provide a password type:

Passw0rd

Press Enter.

Step 6

Once the command has executed, type the following:

sudo apt-get upgrade

Press Enter.

Alert: If you are prompted asking if you wish to continue, type the Y key and press Enter.

Step 7

Once the upgrade process finishes, type the following:

sudo reboot

Press Enter.

Note: This will reboot the machine, and all updates will take effect.
