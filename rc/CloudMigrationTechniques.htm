Exercise 1 - Examining and Performing Cloud Migration Techniques

As more organizations are realizing the benefits of running their operations in the cloud compared to on-prem you will likely find yourself tasked with migrating a physical environment to a cloud provider.

Just as organizational decisions can influence the need to migrate from physical to virtual environments, the same can be said for the need to migrate from one cloud provider to another.

In this exercise, you will examine different cloud migration techniques and then you will create a VHD file of the PLABDM01 system that you are using for this module.

Task 1 - Create a VHD file from a Working Computer

In this task, you will create a VHD file of the PLABDM01 system.

Step 1

Connect to PLABDM01.

Step 2

From the Taskbar, click Microsoft Edge.

Step 3

From the Intranet address bar, type:

disk2vhd download

Press Enter.

Step 4

Click the weblink titled Disk2vhd - Windows Sysinternals | Microsoft Docs.

Step 5

Click Accept on the Cookies message at the top of the page.

Click the Download Disk2vhd weblink.

Step 6

The Disk2vhd.zip will appear in the Downloads folder.

Click the Open file weblink.

Step 7

Click Downloads in the breadcrumb weblink to navigate back to the Disk2vhd folder.

Step 8

Right-click Disk2vhd.zip and select Extract All.

Step 9

Click Extract.

Step 10

A Windows Explorer window will open with the contents that we have just extracted.

Double-click on the Disk2vhd64 application file.

Step 11

From the Disk2vhd License Agreement window click Agree.

Step 12

The Disk2vhd v2.02 application window will appear with the option to select the volumes to include in the vhd we are creating.

Ensure all the Volume checkboxes are ticked.

Click Create.

Step 13

You will see the VHD creation process begin.

Step 14

Once the VHD creation is complete, you will see the message Disk export to VHD completed successfully under the Volume to include section.

Click Close.

Step 15

You will see the Windows Explorer window again and now the PLABDM01 disk image file will be in the Disk2vhd folder.

You have successfully created a VHD image from this machine that can be used as a virtual machine.

Step 16

Close all open windows.
