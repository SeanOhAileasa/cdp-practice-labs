Task 1 - Configure an NTP Client to Synchronize Time with an NTP Server using Control Panel

In this task, the NTP Client will be configured to synchronize time with a preconfigured NTP Server using the Control panel.

Step 1

Connect to PLABWEB01.

Click Start, and type the following:

Control Panel

Select Control Panel.

Step 2

On the Control Panel window, click Clock and Region.

Step 3

In the Clock and Region window, select Date and Time.

Step 4

In the Date and Time dialog box, select the Internet Time tab.

Click the Change settings button.

Step 5

In the Internet Time Settings dialog box, select the Server dropdown and select time.nist.gov.

Click OK.

Step 6

Click OK in the Date and Time dialog box.

Step 7

Close the Clock and Region window.

Step 8

Click Start, type the following:

cmd

Select Command Prompt.

Step 9

In the Command Prompt window, type the following and press Enter:

w32tm /query /peers
Note: From the command’s output, it can be seen that the time server has been updated.

Task 1 - Host Discovery using Linux Commands

Host Discovery is an information gathering technique and is carried out at the beginning of network reconnaissance. Host discovery is carried out using different network assessment tools like nmap, netstat, traceroute etc.

In this task, network Reconnaissance tools will be used to gather more information about hosts on an on-premise network.

Step 1

Connect to the PLABKALI device.

Step 2

Log in to the device using the following credentials:

Username:

root

Password:

Passw0rd
Note: The device may also log in automatically, in which case you can skip this step.

Step 3

Right-click on the Desktop and select Open Terminal Here.

Step 4

In the terminal command, type the following and press Enter.

ifconfig

Note: The IP address, subnet mask and additional network information can be seen from the output. This gathered information can be used to gather more information about the network.

Step 5

In the Terminal window, type the following command and press Enter.

nmap -sS 192.168.0.0/24

Note: After gathering network information, the nmap tool can be used to scan the network for hosts. As can be seen, after the command has been executed, the different hosts with their IP addresses and Operating systems are displayed. Additional information can also be gathered, for example, the open ports on the discovered hosts. Clear the terminal window, typing clear and pressing Enter.

Step 6

In the Terminal window, type the following command and press Enter.

dig practicelabs.com

Note: By using the dig command, more information about the specific domain is gathered. Notice that the IP address for practicelabs.com is displayed.

Task 2 - Information Gathering Tools

Gathering information on different components of the network is essential to ensure the security of it. Different types of information gathering tools are available for Cyber Security professionals, gathering crucial information about the network.

In this task, different information gathering tools will be used to better understand how to gather information about devices on the network.

Step 1

Connect to PLABWIN10.

Step 2

Right-click on Start and select Run.

Step 3

In the Run window, type cmd and click OK.

Step 4

In the Command Prompt window, type the following and press Enter:

arp -a

Note: The arp -a command displays the Address Resolution Protocol table of the device. From the command’s output, the IP and MAC addresses of the devices that this device has been communicating can be seen.

Step 5

Connect to PLABKALI.

Step 6

Sign in to the device using the following credentials:

Username:

root

Password:

Passw0rd

Step 7

Right-click on the Desktop and select Open Terminal Here.

Step 8

In the terminal window, type the following and press Enter:

dnsenum -enum practicelabs.com

Note: The dnsenum command is used to gather more information about an organization’s DNS. After executing the command, the DNS server for the practicelabs.com domain can be seen as plabdc01 with the associated IP address. In the terminal window, type clear and press Enter to clear the terminal window.

Step 9

In the terminal window, type the following and press Enter:

arping -c4 192.168.0.1

Note: The arping command is used to resolve a MAC address from an IP address. The MAC address of the DNS server for the practicelabs.com domain is displayed here. Close the terminal window.
