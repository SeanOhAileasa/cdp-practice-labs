Exercise 1 - Performing a Backup and Restoration of a Virtual Machine

a88527ad-c759-44f1-9711-992dc8f47b03

In this exercise, you will deploy a Virtual Machine in Microsoft Azure, then use this VM to create a snapshot of this machine.

Learning Outcomes

After completing this exercise, you should be able to:

  • Define and Describe Backup types, Backup objects, Backup targets, Backup and Restore policies, and Restoration methods
  • Deploy, backup and restore a virtual machine in Microsoft Azure

Your Devices

You will be using the following devices in this lab. Please power these on now.

af4fa57a-53be-4f51-bf97-9ac3495df239

Task 1 - Deploy a Virtual Machine

In this task, you will deploy a Virtual Machine in Microsoft Azure. This newly created VM will be used in the next task, where you will create a snapshot of this machine.

Step 1

Connect to PLABWIN10.

Step 2

From the Taskbar, select Microsoft Edge.

Step 3

Click Create a resource.

Step 4

From the Create a resource page, under Virtual machine, click the Create weblink.

Step 5

On the Basics tab, under the Project details section, click on the Resource group dropdown and select the pre-populated resource group.

Scroll down to the Instance details section, click into the Virtual Machine name textbox and type:

practicevm01

Step 6

Click the Region dropdown and select (Europe) UK South.

From Images, click the See all images weblink.

Step 7

From the Select an image page, scroll down slightly, so you are able to Microsoft Windows 10 and click the Select dropdown.

Step 8

Select Windows 10 Pro, version 21H2 - x64 Gen 1. 

Step 9

From Size, click the See all sizes weblink.

Step 10

Expand the A-Series v2 dropdown and select A1_v2.

Click Select.

Step 11

From the Administrator account section, click into the Username textbox and type:

plabadmin

In the Password textbox, type:

Passw0rd1234

Type the password again in the Confirm password textbox.

Step 12

From Public inbound ports, ensure Allow selected ports is selected

Also, ensure that the Select inbound ports dropdown is set to RDP (3389).

Step 13

Tick the Licensing checkbox.

Step 14

Click the Tags weblink from the top menu. 

Step 15

Click in the Name textbox and type: 

Practice VM 01

Click in the Value textbox and type: 

Module22

Click Review + create.

Step 16

You may see a Validation Failed message. This is a bug within Microsoft Azure.

To rectify the issue, click < Previous.

Step 17

Click Review + create.

Step 18

Wait until you see the Validation passed message.

Then, click Create.

Step 19

The Overview page of the virtual machine resource will load, and the Deployment is in progress information will be displayed.

Step 20

Once the deployment has completed, a green tick icon will display.

Click the Home weblink.

Keep PLABWIN10 in its current state for this next task.

Task 2 - Create a Snapshot of Virtual Machine

In this task, we will create a snapshot of the Virtual Machine we created in the previous task. This snapshot is a backup of the VM disk and can be used to deploy other VMs.

Step 1

Ensure you are connected to PLABWIN10 and the Azure portal page is open.

From the Azure homepage, click Create a resource.

Step 2

Click in the Search resources, services and docs textbox and type:

snapshot

From the results, select Snapshot.

Step 3

From the Snapshot page, click Create.

Step 4

On the Basics tab, under the Project details section, click on the Resource group dropdown and select the pre-populated resource group.

Scroll down to the Instance details section, click into the Name textbox and type:

practicevm01_snapshot

Step 5

From the Source disk dropdown, select the VM that you created practicevm01[…]

Click Review + create.

Step 6

Click Create.

Step 7

You will now see the Deployment is in progress webpage displayed. Wait for this to finalize.

Step 8

Once the deployment has completed, a green tick icon will display.

Click Go to resource.

Step 9

Notice that the Snapshot resource page shows the details of this newly created snapshot.
